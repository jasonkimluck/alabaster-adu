# GitHub Pages Setup

This repository is prepared for GitHub Pages using the `docs/` folder.

## GitHub settings

1. Open the repository on GitHub.
2. Go to `Settings > Pages`.
3. Under `Build and deployment`, choose `Deploy from a branch`.
4. Set:
   - `Branch`: `codex/alabaster-site-publish`
   - `Folder`: `/docs`
5. Save.

## Published path

- Main site: `/docs/index.html`

## Important note

Only the public marketing site should be published through GitHub Pages. Keep any admin/dashboard files outside the published `docs/` folder.
