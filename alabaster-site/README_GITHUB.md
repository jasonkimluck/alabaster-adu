# GitHub Upload Notes

This folder is the static website that is safe to publish to GitHub Pages or another static host.

## Included

- `index.html`
- `css/`
- `images/`
- `js/`

## Be aware

- Only publish the public marketing site from this folder.
- Do not commit private Google Sheet IDs, service account keys, local admin files, or any `.env` files into the repo.

## Suggested repo shape

If you only want to host the marketing site, publish the public site assets only and keep any admin/dashboard files in a local-only folder outside the published structure.
